# Assets do App

## Ícones e Imagens Necessárias

Você precisa adicionar os seguintes arquivos nesta pasta:

- `icon.png` - Ícone do app (1024x1024px)
- `adaptive-icon.png` - Ícone adaptativo Android (1024x1024px)
- `splash.png` - Tela de splash (2048x2048px)
- `favicon.png` - Favicon (48x48px)

## Logo ULTRAIPTV

O logo é renderizado via código no app, mas você pode adicionar uma imagem aqui se preferir.

## Fontes

Adicione a fonte SpaceMono-Regular.ttf na pasta `fonts/` se quiser usar fontes customizadas.
